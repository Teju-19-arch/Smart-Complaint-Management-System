# Smart-Complaint-Management-System
A web platform where users can report issues (complaints) and track their resolution status.
 # Project Overview
    The Smart Campus Complaint Management System is a web-based platform that allows students and staff to register complaints related to campus facilities such as electricity, internet, water supply, hostel issues, and classroom maintenance.
>   The system ensures that complaints are tracked, assigned to responsible authorities, and resolved efficiently.
# Problem Statement
In many campuses, students report issues through informal methods such as messaging or verbal communication. This leads to:
- Complaints getting lost
- No tracking of complaint status
- Slow resolution
- Lack of accountability
# Proposed Solution
Our system provides a centralized digital platform where users can:
- Submit complaints easily
- Track complaint status
- Receive updates on resolution
- Improve campus maintenance efficiency
 # 🚀 Features
- Student complaint registration
- Upload photo of the problem
- Category based complaint submission
- Complaint tracking system
- Admin dashboard for complaint management
- Complaint status updates
- Image proof for faster issue verification

# ⚙ System Workflow

1. User opens the login page
2. User selects login type:
   - Student
   - Management
3. User enters their ID and password
4. If Student logs in:
   - Can submit complaints
   - Upload photo of the issue
   - Track complaint status
5. If Management logs in:
   - Can view all complaints
   - Can update complaint status
   - Can mark issues as resolved
